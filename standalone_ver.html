Simply Navigation

Lightweight simple navigation built with flexbox and vanilla JavaScript

This is the stand-alone version - everything needed is in this file

This is just a placeholder for an actual page, to show off this navigation component. This navigation solution is built entirely using Flexbox and vanilla JavaScript. Below are some key takeaways.

  • No dependencies. Built using Flexbox & vanilla JavaScript only.
  • Fully responsive.
  • Mobile menu closes when selecting a link, try it. Open the mobile menu and click on the 'Last Section'
  • Easy close mobile menu is closable via entire top row and right side page overlay

Customizable

  • Breakpoint variable - default at 800px
  • Sticky navigation bar available
    • Toggle Sticky Nav is Off
    • Note: Sticky navigation bar requires padding and/or margin on the parent or adjacent element. For example, on this page the adjacent main HTML element has a top margin of 6.5rem (nav height is 4.5rem)
